# 2. 옷장

- 신규 의류 등록 버튼
    - 신규 의류 등록 화면으로 이동하는 탑바의 + 버튼 제공
- 옷 검색 기능
    - 텍스트로 옷 이름, 브랜드 검색
- 옷 필터 기능
    - 카테고리를 통한 필터 기능
        - 전체, 아우터, 상의, 하의, 원피스, 신발, 가방, 액세서리, 운동복, 기타
- 옷 정렬 기능
    - 상태 순 (깨끗한 옷 먼저)
    - 상태 순 (더러운 옷 먼저)
    - 마지막 착용일 순 (DESC)
    - 마지막 착용일 순 (ASC)
    - 자주 입는 옷 (착용 빈도 수
    - 오래된 순 (구매일 기준)
    - 새로운 옷 순 (구매일 기준)
- 옷 리스트 (타일 뷰)
    - 옷 타일
        - 옷 사진
        - 착용 횟수
            - `세탁 후 착용 횟수 / 최대 착용 횟수`
                - 최대 착용 횟수는 옷 등록 및 수정 시 지정한 숫자
                - 최대 착용 횟수가 없을 경우, 그냥 세탁 후 착용 횟수만 표시
        - 마지막 세탁일
            - 해당 의류의 마지막 세탁일 표시 `월/일`
            - 마지막 세탁일이 없을 경우, `-` 표기
        - 착용 횟수가 0회일 경우, 깨끗함 라벨 표기
        - 세탁 후 착용 횟수 ≥ 최대 착용 횟수 일 경우, `세탁 필요` 라벨 표시

## 2-1. 신규 옷 등록

- 의류 사진 등록
    - 앨범 / 촬영 기능 제공
- 의류 명칭 (필수)
    - 텍스트 필드 제공
- 브랜드
    - 텍스트 필드 제공
- 세탁 주기 설정
    - -, + 버튼을 통한 숫자 증감 필드
        - 0으로 설정할 경우 `-` 표기
            - 0은 최대 착용 횟수 미 설정과 같음
- 카테고리 설정 (필수)
    - 아우터, 상의, 하의, 원피스, 신발, 가방, 액세서리, 운동복, 기타
- 세탁 방법
    - 1택
        - 드라이 클린
        - 머신
        - 핸드 워시
- 구매일
    - 캘린더 날짜 선택 입력 창

등록하기

- 등록하기 버튼을 통해 등록하면 해당 옷이 즉시 등록되어 “옷장” 탭에서 확인 가능

## 2-2. 옷 상세

- 탑바 우측 상단에 휴지통 모양 삭제 기능 제공
    - 휴지통 아이콘 버튼 클릭 시, 삭제하시겠습니까? 확인 창 발생 이후 확인 시 삭제
- 상세 내용
    - 옷 이미지
    - 브랜드 표기 라벨
    - 이름
    - 카테고리, 세탁 방법
    - 세탁 후 착용 횟수
        - 착용 횟수 / 최대 착용 횟수
    - 총 착용 횟수
        - 구매 이후부터 착용한 횟수
    - 마지막 세탁일
    - 구매일
- 착용 기록하기
    - 착용 기록하기 버튼 클릭 시, 아래와 같은 입력 창 표기
        - 착용 날짜 (현재일 Default)
            - 클릭 시, Date picker를 통한 날짜 선택 가능
        - 텍스트 입력 필드
            - ex. 평일 출근
    - 최종 “기록” 버튼을 통해 해당 옷의 착용 횟수 증가, 마지막 착용일 업데이트
    - 해당 옷의 착용 히스토리 기록
- 세탁 바구니에 담기
    - 세탁 바구니에 해당 옷 표기하도록 처리
    - 세탁 바구니에 담긴 옷이면 해당 버튼 딤 처리
- 수정
    - 해당 옷 정보 수정 화면 이동
- 착용 히스토리
    - 해당 옷의 착용 기록하기를 통해 기록한 내용 표기
        - 최신 순부터 아래로 (Desc)
    - 해당 옷의 세탁 기록도 같이 표기됨

# 3. 세탁

세탁 바구니

- 세탁 바구니에 담긴 옷들 표기 (리스트)
    - 옷 이미지
    - 이름
    - 카테고리, 세탁 방법
    - 착용 횟수 / 최대 착용 횟수
- 바구니에 담긴 옷을 클릭하여 선택
- 선택 항목 세탁 완료 처리 버튼
    - 옷들을 다시 착용 횟수 0으로 리셋하고, 깨끗한 상태로 처리
    - 세탁 바구니에서 제거
